# Controle Produtos
Um repositório para um sistema básico de controle de produtos


O sistema é feito em Django, apenas por curiosidade e na tetativa de usá-lo no dia a dia para um controle melhor do estoque.

# 1️⃣ Arquivos necessários

Para isso ative o venv de sua máquina com o comando:

``.\venv\Scripts\activate``

Depois rode o seguinte código:

``pip install requirements.txt``

Nesse arquivo está tudo que o projeto precisa para funcionar da melhor forma.
